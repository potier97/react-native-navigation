---

Controls the top bar background styling.

```js
const options = {
  topBar: {
    scrollEdgeAppearance: {
        background: {}
    }
  }
};
```

### `color`

Set the background color.

| Type  | Required | Platform |
| ----- | -------- | -------- |
| Color | No       | Both     |


### `translucent`

Allows the Scroll Edge Appearance to be translucent (blurred).

| Type    | Required | Platform |
| ------- | -------- | -------- |
| boolean | No       | iOS      |
